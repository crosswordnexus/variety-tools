Going Too Far

Instructions

Use this tool to transform a standard crossword into a "Going Too Far" variety puzzle.

  • Mode 1 (Quote-based): Upload a standard crossword file (.puz) and enter a quote in the textarea. Black squares will be converted to gray squares filled with the quote letters.
  • Mode 2 (iPUZ-based): Upload an iPUZ file (e.g., from Ingrid). The tool will interpret circles as gray squares, remove bars, and update word/clue lists.
